============ 小白简易-底部选股 ==================
		1.底部票：几年未涨+几年未涨stable(安全保证)
		2.基本面：业绩OK+分红OK(安全保证)
		3.技术面：均线排列+一阳穿N线（概率大增）
		4.板块筛选：根据市场风口题材选择板块内个股（概率大增）
		5.定增和大宗：如果有，可以窥探主力大概成本
		
		最终：根据个人喜好去选择操作标的
		
		
		
		============ 选股页 ==================
		流通:	是指可流通的筹码，未解禁的筹码是不可以交易的。
		活筹:	是活动筹码，在现有规则下，持股5%的股东减持是需要提前公告的，不公告直接减持算违规，当然也有股东选择违规抢跑。
			所以正常情况下5%的股东不算活动筹码，所以还要在流通筹码中除去这些死筹，当然具体还要看个股的公告，是否有股东减持。
		股东人数(少)： 股东人数一般代表散户账户数，在小票中尽可能的选择股东人数少的股票，但是这个数据披露不及时，非常滞后，只能参考。与同等股份或股权结构下可进行参考比较。
		人均持股(高)： 股东人数人少，平均持股就很高，本系统计算规则：流通股份减去10大股东后的数，然后再来除以股东人数得到数据。只能参考。与同等股份或股权结构下可进行参考比较。
		变化：  是股东人数在3年中(散户很少拿3年这么久的)的最高人数到最新公布的人数然后算出的百分比。数值0：代表最新公布的人数是三年新高很多，人多需要谨慎。
			  负值越大，说明散户割肉的很多，散户割肉多，那主力就可能拿货多。数据披露不及时，参考数据，但可以最近的根据走势和成交量进行估算。
		几年未涨： 用来判断当前股价是否在底部：根据5年的的前复权数据来判断有多少年没有大涨了。每周五计算一次，所以计算基础是上周五的收盘价。参数填写2-5
		几年未涨stable： 因为不同的股票走势完全不同，所以stable的算法比较偏严一些：股价K线的波动率相对没有那么大，波动稳定一些。参数填写2-5
		底部增发： 已经实施的增发，用来判断增发的筹码是否是上市公司相关利益方，自己人。一般在K线底部增发，后期拉高获利的概率较高。
		低于增发价： 当前收盘价是否低于增发价。一般与底部增发一起配合使用，尽量保证你的成本比别人低，安全！注意：需要查询是否在分红的时候有转送股操作！！！
		近5年不亏:   属于基本面保证，5年归属业绩不亏，表示公司运营稳定，暴雷概率小。
		近5年分红：属于 基本面保证，5年分红，表示公司5年都拿真金白银去分红，暴雷概率更小。《近5年不亏》和《近5年分红》一起使用筛选更好！
		国企：  国企相对于民企可能更加有安全感一些。
		大宗：  统计1年或者最近2个月的交易额，侧面分析筹码活跃度，资金活跃度，筹码异动等，因为很多时候是主力行为。具体要看个股的公告以及个人经验分析。单位是亿
		股票代码： 可以输入多个股票代码，以英文逗号,分割开，然后比较，选择最优的个股。
		概念id：   这是用的同花顺的概念id，比如银行的概念是881155，用于板块或者概念的选股以及对比。怎么获取概念id？同花顺打开个股->盘口->所属板块->行业和概念，点开具体的概念获取id
		板块名称搜索： 支持中文模糊搜索板块或者概念，推荐使用id更加精准。
		基本面-避雷区：简单的基本面分析！一般看下资金现金流，应收帐等，具体个股需要个人的财报分析能力和判断能力。《近5年不亏》和《近5年分红》一起使用筛选一般问题不大。
		标签-拉升吸筹：股价拉升，但股东人数大幅减少
		标签-业绩较牛：指业绩同比大幅增长
		标签-白马走势：大市值品质沿均线上升
		标签-小而美：小票，业绩增长还不错
		标签-高质押：高质押是一般双刃剑，可能爆仓导致股价暴跌，也可能大股东会组织拉升股价消除平仓风险。不建议做，总之谨慎。
