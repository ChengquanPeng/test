其他说明:

应付利息:如果较高，公司在大量有息借钱，关联到货币资金和未分配利润。如果货币资金和未分配利润较高，明明有钱为什么借钱？
存货资产:库存变现能力弱
商誉:商誉减值影响扣非净利润，占比太高不行
货币资金/交易性金融资产:相当于可用现金
流动负债:短期负债
应付账款:话语权高，用供应/合作商的钱经营，但如果现金流解决不了应付账款，净资产低于应付账款就会破产清算
应收账款:话语权低，是否自己贴钱在干活，同行业比较
应收款占比:应收账款/流动资产合计
